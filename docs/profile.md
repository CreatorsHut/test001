# 프로필 페이지 (Profile Page)

## 목적
로그인한 사용자가 자신의 개인 정보, 설정, 서비스 이용 현황 등을 확인하고 관리할 수 있도록 합니다.

## 주요 섹션
1.  **사용자 정보 표시**:
    *   프로필 사진 (선택 사항)
    *   이름/아이디
    *   이메일 주소
    *   연락처
2.  **정보 수정 기능**:
    *   개인 정보 (이름, 이메일, 연락처 등) 수정 폼
    *   비밀번호 변경 기능
3.  **서비스 이용 현황 (선택 사항)**:
    *   구독 정보, 결제 내역, 이용 중인 스마트팜 솔루션 현황 등
4.  **설정 (Settings)**:
    *   알림 설정
    *   언어 설정
    *   개인정보 공개 설정 등
5.  **계정 관리**:
    *   계정 탈퇴 기능
    *   로그아웃 버튼
