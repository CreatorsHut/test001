# 회원가입 페이지 (Sign Up Page)

## 목적
사용자가 스마트팜 서비스 이용을 위한 계정을 생성할 수 있도록 합니다.

## 주요 섹션
1.  **회원가입 폼**:
    *   사용자 이름 (또는 아이디)
    *   이메일 주소
    *   비밀번호 (비밀번호 확인 포함)
    *   선택적 정보 (예: 전화번호, 농장 규모 등)
2.  **개인정보처리방침 및 이용약관 동의**:
    *   체크박스 및 링크
3.  **회원가입 버튼**:
    *   폼 제출 버튼
4.  **소셜 로그인 옵션 (선택 사항)**:
    *   Google, Kakao, Naver 등 소셜 계정을 이용한 간편 가입
5.  **로그인 페이지로 이동 링크**:
    *   이미 계정이 있는 사용자를 위한 링크
